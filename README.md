# 🤖 ChatCBC-Bot

A high-performance AI Chatbot built with **Python**, **Tkinter**, and **Groq (Llama 3)**. 

## 🚀 Features
* **Lightning Fast:** Powered by Groq's LPU inference.
* **Desktop Ready:** Compiled `.exe` for easy use on Windows.
* **Secure:** Never hardcodes API keys; saves them locally in `config.txt`.
* **Multi-threaded:** UI stays smooth while the AI is thinking.

## 🛠️ How to Use
1. Download the `ChatCBC-Bot.exe` from the [Releases](your-link-here) section.
2. Run the application.
3. When prompted, enter your **Groq API Key**.
4. Start chatting!
